Introduction

IT Mill Toolkit allows easy creation of first-class web user interfaces for Java-based programs. You can program your user interfaces in Java with our easy-to-learn API; Toolkit will then render it as a stunning AJAX-based single-page web application.

Supports Internet Explorer 6-7, Firefox 1.5-2, Opera 9, Safari 2 and many others.
