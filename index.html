Introduction

IT Mill Toolkit allows easy creation of first class user interfaces (UI) for Java-based programs. UI can be programmed in Java with easy to learn API and the toolkit takes care of creating stunning AJAX-based UI.

AJAX user interfaces support Internet Explorer 5.5-7, Firefox 1-2 and Safari 2. Older browsers are supported by automatically providing traditional web-based UI.

Demo

To get quick feel of the toolkit, execute demo/start.bat on Windows or demo/start.sh on UNIX platform. You now have demo running in http://localhost:8888/.
