Introduction

IT Mill Toolkit allows easy creation of first class web user interfaces (UI) for Java-based programs. UI can be programmed in Java with easy to learn API and the toolkit takes care of creating stunning AJAX-based UI.

AJAX user interfaces support Internet Explorer 6-7, Firefox 1.5-2 and Safari 2. Older or limited browsers are supported by full page HTML rendering mode.
